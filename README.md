# BannerView
A banner view implemented by using RecyclerView for android

### Using
```Groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

dependencies {
    implementation 'com.github.alexxxdev:BannerView:-SNAPSHOT'
}
```
